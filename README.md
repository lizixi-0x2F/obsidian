# obsidian
通过Arduino uno复刻Pong游戏
>Wiring
>> Uno            SSID1306
>>>*  5V <---------> VCC
>>>*  GND <--------> GND
>>>*  A5  <--------> SCL
>>>*  A4 <----------> SDA
>>* Uno            upbutton
>>>* D2  <--------> +
>>>*  GND <--------> GND
>>* Uno            downbutton
>>>* D4  <--------> +
>>>*  GND <--------> GND
>>* Uno            beep
>>>*  VCC <----------> D10
>>>*  GND <-----------> GND
